# To-Do List Manager

A simple To-Do List Manager application built with HTML, CSS, and JavaScript.

## Features
- Add tasks to the list
- Mark tasks as completed
- Delete tasks
- Mark all tasks as completed
- Filter tasks by status (All, Completed, Pending)

- HTML

- JavaScript
